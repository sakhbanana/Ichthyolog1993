# **App Name**: GroupChat

## Core Features:

- User Authentication: Secure user authentication system for login and registration.
- Group Chat: Real-time messaging for a group of 30 users.
- Photo Upload: Ability to upload and share photos within the group.
- Video Upload: Ability to upload and share videos within the group.
- Message Storage: Persistent storage of messages, photos, and videos.
- Push Notifications: Real-time push notifications for new messages.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to create a modern and sophisticated feel.
- Background color: Light grey (#EEEEEE) to provide a clean and neutral backdrop.
- Accent color: Blue (#2196F3) to highlight interactive elements and call-to-actions.
- Body and headline font: 'PT Sans', a modern, clean sans-serif font suitable for both headings and body text.
- Use flat, line-based icons for a clean and modern aesthetic.
- Clean, intuitive layout with clear visual hierarchy.
- Subtle transitions and animations to enhance user experience.
